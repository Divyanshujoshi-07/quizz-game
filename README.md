# 🚀 Advanced Tech Quiz Game

This is a responsive and interactive quiz web application designed to test technical knowledge across multiple programming domains including Python, C, HTML, CSS, JavaScript, and Java.

## ✨ Features
- ⏱️ Timer-based questions (15 seconds per question)
- 📊 Real-time score tracking
- 🧠 50+ advanced interview-level MCQs
- 📱 Fully responsive design (mobile + desktop)
- ⚡ Dynamic question rendering using JavaScript
- 🎯 Clean and modern UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📚 Topics Covered
- Python (OOP, memory, generators)
- C (pointers, memory management)
- HTML (semantic elements, DOM)
- CSS (flexbox, grid, specificity)
- JavaScript (closures, promises, async)
- Java (OOP, JVM, multithreading)

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
